STEP 1: Register GitHub OAuth App (2 minutes)
   → Read: OAUTH_VISUAL_GUIDE.md
   → Go to: https://github.com/settings/developers
   → Click: New OAuth App
   → Fill in: 4 fields (see form template in OAUTH_VISUAL_GUIDE.md)
   → Copy: Client ID and Client Secret

STEP 2: Create Supabase Account (2 minutes)
   → Go to: https://supabase.com
   → Create free project
   → Copy: PostgreSQL connection string

STEP 3: Get OpenRouter API Key (1 minute)
   → Go to: https://openrouter.ai
   → Sign up
   → Copy: API key

STEP 4: Configure .env (2 minutes)
   → cp .env.example .env
   → Fill in: GitHub credentials, DB URL, API key
   → Generate: JWT_SECRET and TOKEN_ENCRYPTION_KEY (openssl commands provided)

STEP 5: Install & Run (5 minutes)
   → npm install (backend + frontend)
   → docker-compose up --build
   → Open: http://localhost:3000
   → Click: "Sign in with GitHub"
   → Test: Submit a prompt!
